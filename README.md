# logs

Open a terminal, either from the launcher or by pressing `Ctrl` + `Alt` + `T` on the keyboard. Enter the commands below:

## All supported distributions:
```
curl -fsSL https://raw.githubusercontent.com/StarLabsLtd/logs/main/star-logs | sudo bash
```

This will create an archive in your downloads folder, for example,
`StarBook_2022-08-07_22-14-40.zip`.

Please send this zip file to support@starlabs.systems
