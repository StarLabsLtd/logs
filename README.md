# logs

Open a terminal, either from the launcher or by pressing `Ctrl` + `Alt` + `T` on the keyboard. Enter the commands below:

## Ubuntu, ZorinOS, Linux Mint, elementaryOS, Debian:
```
sudo apt install curl
curl -s https://raw.githubusercontent.com/StarLabsLtd/logs/main/star-logs | sudo bash
```

## Arch, Manjaro:
```
sudo pacman -S curl 
curl -s https://raw.githubusercontent.com/StarLabsLtd/logs/main/star-logs-arch | sudo bash
```

This will create a zip file in your Downloads folder called, for example, `StarBook_2022-08-07_22-14-40.zip`.

Please send this zip file to support@starlabs.systems
